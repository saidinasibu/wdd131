Site Plan: Discover Guatemala

Site Name

Discover Guatemala

This name was chosen to reflect the purpose of the website,which is to show some touristic places in Guatemala.

Site Purpose

The site will provide detailed information about some touristic places in Guatemala, including their history, location, significance, and visitor information.

Scenarios

  • Where can I find the best historical sites to visit in Guatemala?
  • What are the top tourist attractions in Guatemala?

Color Schema

The selected color schema for the website is:

  • Primary Color: #0073CF (used for the header and navigation background)
  • Accent Color: #FFD700 (used for headings and other accents)
  • Background Color: #FFFFFF (used for the page background)

Typography

The selected typography for the website is:

  • Heading Font: 'Roboto', sans-serif (used for all headings)
  • Body Font: 'Roboto', sans-serif (used for all body text)
